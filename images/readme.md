## Рекомендации по экспорту из Figma:
1. **Логотип 3DLike**: экспортируйте в PNG 2x (244x74px), белый на прозрачном фоне
2. **Девушка с телефоном**: экспортируйте в PNG 2x, высокое качество (рекомендуется 800-1200px по высоте)
3. **Логотипы партнеров**: экспортируйте в PNG 1x, высота 32px, белые на прозрачном фоне
4. **Портфолио**: экспортируйте в JPG, качество 80-90%

## Как экспортировать:
1. Выберите нужный слой в Figma
2. В панели "Export" выберите формат (PNG/JPG)
3. Установите множитель (1x/2x)
4. Нажмите "Export"
5. Сохраните в соответствующую папку проекта
